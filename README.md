## Features

- Product Listing : Browse products in a responsive grid layout.
- Search : Search products with URL-based queries (`?search=product-name`).
- Filters :
  - Category filter
  - Brand filter
  - Price range slider
- Product Details :
  - Dynamic product pages (`/product/[id]`)
  - Product image, description, price, and rating
  - Quantity selection
- Cart :
  - Add / remove products
  - Update product quantity
  - Cart badge synced across pages
  - Cart state persisted using localStorage
- Routing :
  - Dynamic routing with Next.js App Router
  - Custom `not-found` page handling
- Responsive UI :
  - Optimized for desktop, tablet, and mobile screens
